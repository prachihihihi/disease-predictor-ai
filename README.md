# Disease Predictor AI

An AI-based system that predicts probable diseases based on user-input symptoms, using a Naive Bayes classifier for probabilistic reasoning under uncertainty.

## Project Structure
- `data/` — Training and testing datasets (symptom-disease dataset, Kaggle)
- `src/` — Source code (data processing, model, interface)
- `docs/` — Report, architecture diagrams, supporting documents
