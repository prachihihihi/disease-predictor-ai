# Disease Predictor AI

An AI-based system that predicts probable diseases based on user-input symptoms, using a Bernoulli Naive Bayes classifier for probabilistic reasoning under uncertainty.

## Project Structure
- `data/` — Training and testing datasets (symptom-disease dataset, Kaggle)
- `src/` — Source code (data processing, model training, CLI interface)
- `docs/` — Report, architecture diagram, evaluation visuals

## How to Run
1. Install dependencies: `pip3 install -r requirements.txt`
2. Train and evaluate models: `python3 src/train_model.py`
3. Run the interactive predictor: `python3 src/cli.py`
